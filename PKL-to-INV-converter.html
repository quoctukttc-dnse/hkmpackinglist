Chuyển Gút hàng → Invoice + Packing List

Upload file gút hàng (.xlsx) → chọn PO → tải về file INV và PKL chi tiết theo size. Giá để trống điền sau.

1 · Tải file gút hàng

Kéo thả file GÚT HÀNG (.xlsx) vào đây, hoặc bấm để chọn file

2 · Thông tin invoice

Mỗi PO sẽ nhận số invoice dạng <base>-01, -02, … theo thứ tự được chọn.

3 · Chọn PO cần xuất

Sheet / POMã hàngMàu PcsCartonsN.WG.WCBMTrạng thái

4 · Xuất file

INV: cột PRICE và TOTAL AMOUNT để trống (điền sau). PKL: chi tiết từng size, kèm số carton, N.W, G.W, CBM.
