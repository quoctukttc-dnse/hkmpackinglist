Chuyển Gút hàng → Invoice + Packing List

Upload file gút hàng (.xlsx) → chọn PO → tải về INV + PKL. Mỗi PO gộp vào 1 sheet (nhiều thị trường xếp chồng). Giá để trống điền sau.

1 · Tải file gút hàng

Kéo thả file GÚT HÀNG (.xlsx) vào đây, hoặc bấm để chọn file

2 · Thông tin invoice

Mỗi PO nhận số invoice dạng <base>-01, -02, … theo thứ tự. Các trường không có trong gút (EAN, prepack no, consolidation…) để trống.

3 · Chọn PO cần xuất

POThị trường (DC)Mã hàng PcsCartonsTrạng thái

4 · Xuất file

PKL: mỗi PO 1 sheet, mỗi thị trường 1 block (header + prepack + ma trận size + colour/size breakdown + shipping marks). INV: mỗi PO 1 sheet, mỗi DC 1 dòng.
